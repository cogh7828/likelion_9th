# 텍스트와 관련된 태그

※ 특별한 경우 아니면 왠만하면 태그 달 것

---

제목 태그 1 ~ 6 : <h1> </h1>  ~ <h6> </h6>

---

본문 태그 

- < p>   < /p>

  > paragraphs; 단락, 문단
  >
  > 줄 바꿈 X

- < /br>

  > 줄 바꿈, 종료 태그 없음
  >
  > empty tag : 요소X 

- < pre>

  > 적은 대로 출력, pre태그보다 보통 p 태그를 사용
  >
  > ※ 들여쓰기도 그대로 출력

- < strong> < /strong>, < em> < /em>

  > 강조
  >
  > 특정 텍스트 bold체, 이텔릭체

- < sub> < /sub> , < sup> < /sup>

  > log_{10}^4
  >
  > e^2

- < ins> < /ins> , < del> < /del>

  > 밑줄, 취소선
